## Core Features

### 1. Bulk CV Upload

- Upload ZIP folder containing multiple CVs
- Support PDF, DOC, DOCX
- Parse all CVs automatically
- Show progress bar during processing

### 2. AI-Powered Analysis

- Each CV gets an ATS score
- Skills extraction
- Experience level detection
- Red flags detection (gaps, job hopping)
- Green flags detection (career growth)

### 3. Job-Based Matching

- Paste job description
- AI ranks all uploaded CVs by fit
- Shows top matches with reasoning
- Filter by minimum match score

### 4. Smart Filters

- Years of experience
- Specific skills required
- Education level
- Location
- Salary expectations
- Employment history quality

### 5. Bulk Actions

- Export shortlist to CSV/PDF
- Send bulk personalized emails
- Reject candidates with AI-written emails
- Schedule interviews

### 6. Candidate Insights

- Personality analysis
- Cultural fit prediction
- Career trajectory analysis
- Skills gap analysis vs job requirements

### 7. Team Collaboration

- Multi-user accounts
- Comments on candidates
- Rating system
- Role-based permissions

### 8. Analytics Dashboard

- CVs analyzed this month
- Top skills in your candidate pool
- Salary benchmarks
- Time saved metrics
- Hiring funnel analytics

### 9. Communication Tools

- Email templates library
- Auto-follow-up sequences
- Interview invitation automation
- Rejection letter generator

### 10. Integrations (Future)

- Export to popular ATS (Workday, Greenhouse, Lever)
- Google Calendar for interviews
- LinkedIn candidate lookup
- WhatsApp Business API
